# 📊 Incident Analytics Platform

> Universal incident analytics dashboard that works with ANY company's ticket data — JIRA, ServiceNow, Zendesk, Freshdesk, or any CSV export.

## 🚀 Live Demo
👉 **[Try it Live](https://vidyachoki.github.io/incident-analytics-platform/)**

## ✨ Features
- Upload any CSV ticket file — auto-detects columns
- 6 KPI cards (Total, Resolved, Open, SLA Issues, P1 Count, Avg Resolution)
- AI-powered insights with recommendations
- Monthly ticket trend chart
- Priority distribution, Status breakdown, Issue type split
- SLA Breach Detector — highlights breached and at-risk tickets
- Team / Assignee performance table with resolution rates
- Full ticket table with filters (Priority, Status, Assignee)
- Export report as CSV
